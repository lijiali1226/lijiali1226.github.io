软件安装

Homebrew

常用命令
brew install wget
brew uninstall wget
brew search xxx
brew list 列出已安装的软件
brew update
brew info 显示应用的信息
brew deps 显示包依赖
Homebrew将应用独立安装在类似/usr/local/Cellar/wget之类的地方，在/usr/local/bin下面建立一个到具体安装路径的符号链接

brew cask

安装brew cask
brew install caskroom/cask/brew-cask
常用命令
brew cask install
brew cask uninstall
brew cask search xxx
brew cask info 显示应用的信息
默认安装在/opt/homebrew-cask/Caskroom/下，会在home目录中的Applications文件夹中创建一个类似快捷方式的替身

影音娱乐

F.lux

调节色温，可以根据经纬度设置地点，跨平台，windows也可以装一个，保护视力

Boom

增大音量和音效

浏览器用html5模式看视频

safari，偏好设置->高级，勾选“在菜单栏显示开发菜单”。
显示开发菜单之后，开发->用户代理，选ipad。
把浏览器伪装成移动设备，就可以不安装flash插件看视频了。
注意：有的网站只能提供前几分钟预览，非要你在ipad上安装该应用才能看完，瞬间印象掉到谷底。

mpv

brew install mpv
brew cask install mpv
一开始用的brew，加链接也不起作用，每次都要在命令行里面开启，后来用brew cask加了个壳，方便使用。
将brew cask生成的快捷方式复制到系统的application文件夹，设置为默认打开视频文件的软件。
mpv项目github地址

默认快捷键

  • Space 暂停/播放切换
  • # 循环切换音轨
  • f 切换全屏状态
  • 9 音量减 2
  • 0 音量加 2
  • [ 0.9091 倍慢速播放
  • ] 1.1 倍快速播放
  • { 0.5 倍慢速播放
  • } 2 倍快速播放
  • Backspace 重置为正常播放速度
  • q 退出

主要配置文件(mpv.conf)

~/.config/mpv/mpv.conf
官方文档 mpv options

快捷键配置(input.conf)

~/.config/mpv/input.conf
官方文档 mpv keybindings

程序员工具

iTerm 2

安装
brew cask install iterm2
有很多实用的特性，terminal替代物

其他选项
用maximum-awesome一次性解决，包括MacVim、iTerm 2、tmux和各自的配置。
maximum-awesome项目github地址

Zsh

前提条件
unix类操作系统，需要安装zsh，git，curl或者wget
cat /etc/shells 查看已有的shell

安装
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
或者
sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

配置
oh-my-zsh项目github地址
