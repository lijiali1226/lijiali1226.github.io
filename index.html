SQL Tips

2016-11-29
database
  1. 用sql beautifier 格式化sql，便于自己和他人阅读
  2. 选择字段的时候把id一类的主外键放在前面
  3. 选择字段的时候将业务上有关联的字段写在一起，比如产品部门的层级结构，或者和财务相关的字段等等
  4. 写where条件的时候把表关联条件写在前面，比如a.id=b.aid(+)，然后再写其他的限制条件
  5. 写sql之前先想好主表是哪一个，放在所有表的最前面，和其他表做连接的时候想好记录数的变化
  6. 尽量用有意义的表别名，不要老用abcd之类
