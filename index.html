概念

常识

TPS——Transaction Per Second 每秒事务处理量
IOPS——Input/Output Operations Per Second 每秒进行读写（I/O）操作的次数，衡量随机访问的性能
QPS——Query Per Second 每秒查询数
RT——Response Time 响应时间
吞吐量——Throughput 系统在单位时间内处理请求的数量
TPC——Transaction Processing Performance Council 一系列事务处理和数据库基准测试的规范

MySQL Tips

2016-04-18
database

1.Linux下的MySQL，数据库名、表名、变量名默认是严格区分大小写的。
让MySQL不区分表名大小写：
　a.切换到root用户，修改/etc/my.cnf
　b.在[mysqld]下加入一行：lower_case_table_names=1 （0区分，1不区分）
　c.重新启动数据库 /etc/init.d/mysqld restart
